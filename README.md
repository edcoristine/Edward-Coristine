National Design Studio

National Design Studio is focused on improving how people interact with public services.

The work starts with a simple idea: public technology should respect people’s time. That means fewer unnecessary steps, clearer language, faster flows, and interfaces that feel understandable from the first screen.

My work at NDS is centered around turning that idea into usable software — from design systems and service prototypes to privacy-focused tools that run directly on the user’s device.

Featured Work
Rampart

Rampart is a local-first privacy model that helps detect and redact personally identifiable information before text leaves the user’s device.

The goal is simple: users should be able to interact with modern AI and web services without accidentally sharing sensitive details.

Rampart runs in the browser and combines deterministic redaction rules with lightweight machine learning for fast, practical privacy protection.
